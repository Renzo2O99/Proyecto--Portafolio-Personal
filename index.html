Sitio Web - Cafetería

El sitio web de la cafetería está diseñado para proporcionar una experiencia de navegación agradable y fluida, destacando la calidad de sus productos y el proceso de elaboración del café.

Además de la estructura, se implementó el Microprocesador SASS para la elaboración y optimización del código CSS. También se utilizaron otras tecnologías, como NPM para el manejo de dependencias y Gulp para la automatización de tareas repetitivas durante el desarrollo.
